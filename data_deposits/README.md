# data_deposits

Staging area for **GEO (Gene Expression Omnibus) submission** of the raw
sequencing data underlying this project.

## Contents

- `GEO_seq_template.xlsx` — the GEO SOFT submission template populated
  with sequencing-run metadata, sample annotations, and per-file MD5s.

## Status

Submission is in preparation. Once the GEO accession is issued it will be
linked from the top-level [`README.md`](../README.md) and from the
manuscript.
